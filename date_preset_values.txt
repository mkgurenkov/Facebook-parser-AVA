today - Данные за сегодняшний день (с полуночи до текущего времени).
yesterday - Данные за вчерашний день (с полуночи до полуночи).
last_3d - Данные за последние 3 дня.
last_7d - Данные за последние 7 дней.
last_14d - Данные за последние 14 дней.
last_28d - Данные за последние 28 дней.
last_30d - Данные за последние 30 дней.
last_90d - Данные за последние 90 дней.
last_month - Данные за прошлый календарный месяц (например, с 1-го по 31-е октября).
this_month - Данные за текущий календарный месяц (например, с 1-го ноября до текущей даты).
this_week_mon_today - Данные за текущую неделю, начиная с понедельника до текущей даты.
this_week_sun_today - Данные за текущую неделю, начиная с воскресенья до текущей даты.
last_week_mon_sun - Данные за прошлую неделю (с понедельника по воскресенье).
last_week_sun_sat - Данные за прошлую неделю (с воскресенья по субботу).
this_quarter - Данные за текущий календарный квартал.
last_quarter - Данные за прошлый календарный квартал.
lifetime - Данные за весь период существования кампании/объявления.
last_year - Данные за прошлый календарный год.
this_year - Данные за текущий календарный год
